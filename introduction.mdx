# Welcome to Public Records API Documentation

The Public Records API is an AI-powered service that provides criminal record evaluation through multiple decision engines. The API orchestrates processing through XML transformation, compliance rule evaluation, and machine learning models to deliver comprehensive criminal record assessments.

## Architecture

The Public Records API orchestrates a multi-step workflow:

1. **XML-Transformer** - Transforms raw XML criminal records into structured JSON (sequential)
2. **Compliance-Engine** - Evaluates records against compliance rules (parallel)
3. **SLM Engine** - Evaluates records through Structured Language Model evaluation (parallel)
4. **T5 Model API** - Provides additional ML-based evaluation capabilities (parallel)
