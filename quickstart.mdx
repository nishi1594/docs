# Quickstart Guide

Get up and running with the Public Records API in just a few minutes.

## Prerequisites

- API key from your account manager
- cURL or your preferred HTTP client
- Basic understanding of REST APIs

## Step 1: Get Your API Key

Contact your account manager to obtain your API key. Your API key will look like:

```
eval-api-key-4d8cebd8baaa9844cc2c5a975b7afa28
```

## Step 2: Exchange API Key for JWT Token

Use your API key to get a JWT token:

```bash
curl -X POST https://awp-ai-eval-framework-cr-dev-859798439223.us-south1.run.app/auth/token \
  -H "X-API-Key: YOUR_API_KEY_HERE" \
  -H "Content-Type: application/json"
```

**Response:**
```json
{
  "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "token_type": "bearer",
  "expires_in": 7200
}
```

Save the `access_token` - you'll use it in the next step.

## Step 3: Evaluate Your First Record

Now evaluate a criminal record:

```bash
curl -X POST https://awp-ai-eval-framework-cr-dev-859798439223.us-south1.run.app/evaluate \
  -H "Authorization: Bearer YOUR_TOKEN_HERE" \
  -H "Content-Type: application/json" \
  -d '{
    "engines": ["CE"],
    "record": {
      "xml": "<ScreeningResults><login><account>05022022</account><username>test</username></login><postResults order=\"12345\"><case><case_number>MJ-05003-TR-0006023-2020</case_number><offense_date>20200310</offense_date><jurisdiction_state>PA</jurisdiction_state><chargeinfo><charge>DRIVING WHILE OPERATOR PRIVILEGES SUSPENDED</charge><disposition>GUILTY</disposition></chargeinfo></case></postResults></ScreeningResults>",
      "search_id": "test-123",
      "search_date": "2024-01-15",
      "applicant_state": "CA"
    },
    "response_format": "detailed"
  }'
```

**Response:**
```json
{
  "search_id": "test-123",
  "correlation_id": "uuid-456",
  "status": "success",
  "engines_called": ["CE"],
  "individual_decisions": {
    "CE": {
      "decision": "PASS",
      "confidence": 0.95
    }
  },
  "data": {...},
  "timing": {
    "xml_transformer_ms": 1234.56,
    "compliance_ms": 567.89,
    "total_ms": 1802.45
  }
}
```

## Step 4: Use Multiple Engines

Try evaluating with multiple engines for comparison:

```bash
curl -X POST https://awp-ai-eval-framework-cr-dev-859798439223.us-south1.run.app/evaluate \
  -H "Authorization: Bearer YOUR_TOKEN_HERE" \
  -H "Content-Type: application/json" \
  -d '{
    "engines": ["SLM", "CE", "T5"],
    "record": {
      "xml": "<ScreeningResults>...</ScreeningResults>",
      "search_id": "test-456",
      "search_date": "2024-01-15"
    },
    "response_format": "short"
  }'
```

This will return a comparison view showing decisions from all three engines.

## Common Issues

### 401 Unauthorized
- **Cause**: Token expired or missing
- **Solution**: Get a new token from `/auth/token`

### 422 Validation Error
- **Cause**: Missing required fields or invalid format
- **Solution**: Check that `search_id`, `search_date`, and `xml` are provided

### 500 Server Error
- **Cause**: Service temporarily unavailable
- **Solution**: Retry the request after a few seconds
